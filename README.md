<h1 align="center"> LyceumBot</h1>
<h4 align="center">Lyceum Bot for viewing schedules in Telegram</h4>

## Идея
Бот для удобного просмотра раписания ученикам МАОУ "Лицей №2".

Все данные парсятся из открытой гугл таблицы с расписанием.

 
В боте можно быстро и удобно посмотреть:

-Расписание своего класса

-Расписание любого учителя

-Расписание любого другого класса

-Расписание звонков

А также поменять вид отображения пар с уроков на пары и наоборот.

## Смысл
Бот написан исключительно для обучающихся в МАОУ "Лицей №2" от бывшего ученика.

Создавался, как продукт рассчитанный на конкретную целевую аудиторию, которым будут пользоваться.

В будующем планирую полностью переписать бота на Aiogram и снова запустить в работу.
